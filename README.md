# Лендинг типографии (учебный проект)

Небольшой одностраничный сайт для учебного задания по управлению производством.  
Проект полностью статический: весь код в одном файле `index.html`.

## Что нужно скачать

Минимально:
- любой современный браузер (Chrome, Edge, Firefox, Яндекс Браузер).

Опционально (для удобной работы):
- Visual Studio Code;
- расширение `Live Server` в VS Code.

Важно:
- интернет нужен только для загрузки шрифтов Google Fonts;
- без интернета сайт тоже откроется, просто шрифт будет стандартный.

## Как запустить

### Вариант 1 (самый простой)
1. Открыть папку проекта.
2. Дважды кликнуть по `index.html`.

### Вариант 2 (через VS Code)
1. Открыть папку проекта в VS Code.
2. Открыть файл `index.html`.
3. Нажать `Go Live` (если установлен Live Server).

## Что уже реализовано

- адаптивная верстка (mobile-first);
- темная тема с акцентами и анимациями;
- плавная навигация по якорям;
- подсветка активного пункта меню при прокрутке;
- accordion-блоки:
  - требования к макетам;
  - этапы контроля качества;
- демо-форма заявки (без отправки на сервер);
- модальное окно подтверждения для формы;
- кнопка `Скачать прайс` как заглушка (показывает модальное сообщение);
- анимации появления блоков при скролле;
- легкий 3D-эффект на hero-блоке (на устройствах с мышью);
- защита от вылезания текста из карточек на разных экранах.

## Структура проекта

- `index.html` — весь HTML, CSS и JS в одном файле.
